# PaisaVasool

**PaisaVasool** is a modern, intuitive personal finance management app built with Jetpack Compose. It helps users track their income and expenses, visualize spending patterns, and maintain a healthy financial balance.
This application is completely developed with AI prompts using Gemini and Antigravity.

## Features

- **📊 Dynamic Dashboard**: Get a quick overview of your total balance and view your most recent transactions.
- **📈 Visual Analytics**: Understand your spending habits with an animated category-wise expense breakdown pie chart.
- **📝 Complete Transaction Management**:
    - Add new income or expense entries.
    - Categorize expenses (Food, Shopping, Transport, etc.).
    - Edit or delete existing transactions easily.
- **🕒 Detailed History**: View a full list of all your past transactions with precise timestamps and amounts.
- **🌙 Dark Mode Optimized**: A premium aesthetic designed for visibility and ease of use in dark environments.
- **🚀 Smooth Navigation**: Instant, fluid transitions between features for a seamless user experience.

## Tech Stack

- **UI**: [Jetpack Compose](https://developer.android.com/jetpack/compose) for a modern, declarative UI.
- **Language**: [Kotlin](https://kotlinlang.org/)
- **Database**: [Room Database](https://developer.android.com/training/data-storage/room) for local data persistence.
- **Navigation**: [Jetpack Navigation](https://developer.android.com/guide/navigation) for a robust single-activity architecture.
- **Architecture**: MVVM (Model-View-ViewModel) for clean code separation and maintainability.

## Installation

1. Clone the repository.
2. Open the project in **Android Studio**.
3. Sync Gradle and run the app on an emulator or a physical device (Min SDK 24).
